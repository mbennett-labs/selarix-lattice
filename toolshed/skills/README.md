# Toolshed: Skills

Model-facing skill prompts. Instructions that teach an agent HOW to think about a task — "how to write a therapist bio in Mike's voice," "how to evaluate a scraped listing for data quality," etc.

Skills are prose markdown files. They get loaded into an agent's context when that agent picks up relevant work.
