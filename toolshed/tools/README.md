# Toolshed: Tools

Reusable code we've written. Scrapers, generators, parsers, schedulers, anything that's built once and used across projects.

Unlike adapters (which wrap external services), tools are our own codebase. They live versioned in git, tested, documented.

## Rules
1. Every tool exists because a real project needed it
2. Wrap on first use, harden on second use, consider it mature on third use
3. Project-specific tools live in the project's repo; shared tools live here
