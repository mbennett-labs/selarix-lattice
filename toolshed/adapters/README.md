# Toolshed: Adapters

Wrappers for external services — APIs, databases, third-party tools. These are NOT things we write from scratch; they are thin adapters around services we already use (OpenRouter, Telegram, AdSense, Stripe, etc.).

An adapter is typically 50-200 lines. The underlying service is someone else's problem; the adapter makes it speak the Lattice's language.

## Required header for every adapter file
- Kind: adapter
- Status: active | experimental | deprecated
- What it does: one sentence
- When to use it
- When not to use it
- Dependencies
- Model compatibility
- Last updated
